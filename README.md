# TicTracker

轻量级 macOS 菜单栏计数工具。快捷键记录，日报提醒，周报汇总。

## 功能

- 菜单栏常驻，点击即用
- 自定义项目分类，按项目计数
- 每个项目可自定义独立快捷键组合（录制式设置）
- 快捷键快速日报弹窗（首个修饰键+0）
- 日期切换，回看和编辑历史数据
- 每日小记，支持 Markdown
- 本周趋势图 — 按项目堆叠彩色柱状图，点击查看明细
- 连续打卡天数
- 日期范围项目统计视图（Swift Charts）
- 每日提醒写日报（支持稍后提醒）
- 下班工作摘要通知（日报提醒后 30 分钟自动推送今日统计）
- 通知交互按钮 — 打开日报、稍后提醒、打开链接、复制周报、查看详情
- 点击 +1 时记录时间戳，日报视图 hover 查看
- 一键复制周报汇总
- 数据导入 / 导出（JSON、CSV）
- Jira Server/DC 集成 — 工单查看、计数、状态流转、自动映射到项目
- Jira 流转自动关联项目计数（每天每工单仅一次）
- RSS 订阅监控（通知支持直接打开链接）
- 开机自动启动

## 系统要求

- macOS 14.0+（Sonoma），macOS 15+ 体验更佳
- Swift 6.0+

## 构建

```bash
# 编译并打包为 .app
bash build.sh

# 启动
open TicTracker.app
```

脚本会自动完成 release 编译、组装 .app bundle、ad-hoc 签名。

## 快捷键

每个项目可在「设置 → 通用 → 快捷键」中单独录制快捷键组合（如 `⌃⇧1`、`⌘⌥A` 等）。
升级时会自动从旧版全局修饰键迁移。

| 操作 | 说明 |
|------|------|
| 点击录制框 → 按下组合键 | 为该项目绑定快捷键 |
| ✕ 按钮 | 清除该项目的快捷键 |
| 首个修饰键+0 | 打开快速日报弹窗 |

## 通知

| 通知类型 | 按钮 | 说明 |
|---------|------|------|
| 日报提醒 | 打开日报 / 稍后提醒 | 每天定时提醒，稍后提醒 15 分钟后再次推送 |
| RSS 新条目 | 打开链接 | 直接在浏览器打开对应 URL |
| 每日摘要 | 复制周报 / 查看详情 | 日报提醒后 30 分钟推送今日统计 |

## 数据存储

数据保存在 `UserDefaults`，包括项目列表、每日计数、时间戳和小记。支持 JSON / CSV 导出备份。
