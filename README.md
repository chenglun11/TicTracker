# TicTracker

轻量级 macOS 菜单栏计数工具。快捷键记录，日报提醒，周报汇总。

## 功能

- 菜单栏常驻，点击即用
- 自定义项目分类，按项目计数
- 全局快捷键 `⌃⇧+数字键` 快速 +1
- 日期切换，回看和编辑历史数据
- 每日小记，支持 Markdown
- 本周趋势图 + 连续打卡天数
- 每日提醒写日报
- 一键复制周报汇总
- 数据导入 / 导出（JSON）
- 开机自动启动

## 系统要求

- macOS 14.0+
- Swift 5.9+

## 构建

```bash
# 编译并打包为 .app
bash build.sh

# 启动
open TechSupportTracker.app
```

脚本会自动完成 release 编译、组装 .app bundle、ad-hoc 签名。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌃⇧1` | 第 1 个项目 +1 |
| `⌃⇧2` | 第 2 个项目 +1 |
| ... | 最多支持 9 个项目 |

## 数据存储

数据保存在 `UserDefaults`，包括项目列表、每日计数和小记。支持 JSON 导出备份。
